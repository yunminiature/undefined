# 🎮 Игра 2048

Веб-приложение с классической игрой 2048, реализованное на современном стеке технологий.

## 📋 Описание игры

**2048** — это однопользовательская головоломка, в которой игрок должен объединять плитки с одинаковыми числами, чтобы достичь плитки со значением 2048.

## 🎯 Правила игры

1. **Управление**: Используйте стрелки ↑ ↓ ← → для перемещения плиток
2. **Объединение**: Когда две плитки с одинаковыми числами соприкасаются, они объединяются в одну с суммой их значений
3. **Цель**: Создать плитку с числом **2048** для победы
4. **Конец игры**: Игра заканчивается, когда поле заполнено и больше нет возможных ходов

## ⚙️ Механика игры

### Основные принципы:
- **Размер поля**: 4×4 клетки
- **Начальные плитки**: 2 плитки со значениями 2 или 4
- **Новые плитки**: После каждого хода появляется новая плитка (90% вероятность для "2", 10% для "4")
- **Подсчёт очков**: Очки начисляются за каждое объединение плиток

### Алгоритм движения:
1. Все плитки сдвигаются в выбранном направлении
2. Соседние плитки с одинаковыми значениями объединяются
3. Объединённые плитки не могут объединяться повторно в том же ходу
4. После хода в случайной пустой клетке появляется новая плитка

### Условия победы и поражения:
- **Победа**: Появление плитки со значением 2048
- **Поражение**: Отсутствие возможных ходов (поле заполнено и нет соседних плиток с одинаковыми значениями)

## 🧠 Логика игры

### Основные функции:
- `move()` — обработка движения плиток в заданном направлении
- `placeRandomTile()` — размещение случайной плитки на пустой клетке
- `checkIsWon()` — проверка условия победы
- `checkIsGameOver()` — проверка условия поражения

### Алгоритм объединения:
```typescript
// Пример логики объединения для строки
const newRow = [...row].filter(v => v !== 0) // Убираем пустые клетки

for (let i = 0; i < newRow.length - 1; i++) {
  if (newRow[i] === newRow[i + 1]) {
    newRow[i] *= 2        // Объединяем плитки
    gainedScore += newRow[i]  // Добавляем очки
    newRow[i + 1] = 0     // Удаляем вторую плитку
  }
}
```

## 🛠️ Техническая реализация

### Ключевые компоненты игры:

#### 1. GameBoardCanvas
- Рендеринг игрового поля через HTML5 Canvas
- Оптимизированная отрисовка с отслеживанием изменений

#### 2. useGameBoard Hook
- Управление состоянием игры
- Обработка клавиатурных событий
- Логика движения и объединения плиток

#### 3. Game Logic (utils/game.ts)
- `generateEmptyBoard()` — создание пустого поля
- `move()` — основная логика движения
- `placeRandomTile()` — размещение новых плиток
- `checkIsWon()` / `checkIsGameOver()` — проверка условий

### Конфигурация игры:
```typescript
export const GAME_CONFIG = {
```
